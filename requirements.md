# 项目：单机游戏推荐站

## 核心需求
搭建一个网站，每周自动抓取 3DM、游侠等国内游戏网站的热门新游，展示给用户。

## 功能模块
1. **数据抓取层 (Scraper)**:
   - 来源 1: 3DM (www.3dmgame.com) - 新游中心/热门榜
   - 来源 2: 游侠网 (ali213.net) - 单机游戏排行榜
   - 输出格式: 统一 JSON 结构
2. **数据聚合层 (Aggregator)**:
   - 清洗数据 (标题，链接，封面图，简介，来源，时间)
   - 去重处理
3. **前端展示层 (Frontend)**:
   - 响应式网页设计
   - 游戏卡片展示 (图片 + 标题 + 简介 + 链接)
   - 来源标签过滤

## 技术栈约束
- **语言**: Python 3.x (爬虫 + 数据处理)
- **前端**: HTML5 + CSS3 + Vanilla JS (无需复杂框架，保持轻量)
- **数据持久化**: 本地 JSON 文件 (后续可扩展数据库)
- **部署**: 静态网页托管 (GitHub Pages / Vercel) 或本地 Python HTTP Server

## 验收标准
1. 运行 `python run_scraper.py` 能生成 `data/games.json`
2. 打开 `index.html` 能看到来自两个网站的游戏推荐
3. 代码有基本注释，结构清晰
